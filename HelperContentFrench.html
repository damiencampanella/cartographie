: pas de prédiction de charge - : charge légère - : charge modérée - : charge importante

Aller Le second indique sa direction

En retard Le troisième indique son retard

Cliquez sur une station pour afficher les horaires des trois prochains passages dans la direction sélectionnée

Partie menu:

L'onglet "Lignes" vous permet de sélectionner les lignes qui vous intéressent

Amenez la souris au-dessus d'une ligne pour masquer les autres lignes

Lignes Masquées Ce bouton permet de griser les autres lignes au lieu de les masquer

Cliquez sur une ligne pour la désélectionner

Ce bouton permet de désélectionner toutes les lignes de la catégorie correspondante


L'onglet "Sélection" vous permet choisir la direction et la variante sélectionnée pour chaque ligne

Cliquez sur une des lignes pour obtenir la liste des variantes et la liste des stations de la variante sélectionnée

Ce bouton permet de changer de direction

Ce bouton permet de désélectionner la ligne

Cliquez sur la variante voulue pour la sélectionner

Amenez la souris au-dessus d'une des stations de la liste pour afficher les informations correspondantes

A droite de chaque station sont affichées les correspondances possibles

Supprimer toutes les lignes sélectionnées Ce bouton permet de désélectionner toutes les lignes


L'onglet "Véhicules" vous permet d'afficher la liste des véhicules ayant un capteur

Utilisez le champ correspondant pour rechercher parmi les véhicules

La recherche peut se faire par nom, par numéro de stock (chiffre entre parenthèses) ou par numéro de capteur

Amenez la souris au-dessus d'un des véhicules de la liste pour afficher les informations correspondantes

Cliquez sur un véhicule pour zoomer dessus

Masquer les véhicules Ce bouton permet de masquer tous les véhicules
